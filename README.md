# SmartCart Customer Segmentation

This project performs customer personality analysis and segmentation for **SmartCart** using a clustering approach. It helps businesses understand their customers' behavior to better tailor marketing strategies.

## 📝 Project Overview
The project involves:
- **Data Preprocessing**: Handling missing values (e.g., in the 'Income' column).
- **Feature Engineering**: Creating new features like `Age`, `Total_Spending`, `Customer_Tenure_Days`, and simplifying `Education` and `Living_With` statuses.
- **Data Analysis**: Visualizing distributions and relationships using Seaborn and Matplotlib.
- **Clustering**: Grouping customers based on their purchasing habits and demographics.

## 🚀 Getting Started

### Prerequisites
